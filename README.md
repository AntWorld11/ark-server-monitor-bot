# ARK Server Monitor Bot

Python Discord bot that monitors server populations using the BattleMetrics API and sends automated alerts.

## Features

* Monitors player counts across multiple servers
* Detects population spikes
* Alerts when servers drop to low population (wipe windows)
* Sends real-time alerts to Discord channels
* Includes commands to check server status

## Commands

* !pop → Check main server population
* !check <server_id> → Check a specific server

## Tech Used

* Python
* discord.py
* BattleMetrics API

## Purpose

Built to automate server monitoring and demonstrate real-time alerting, API integration, and system automation.

## Note

This is a edited version of the bot. Sensitive information such as tokens and IDs have been removed for security.
